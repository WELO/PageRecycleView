# PageRecycleView
可偵測高度修改列數的PageRecycleView<br/>
主要使用<br/>
pageRecycleView<br/>
-OnDataChangeListener<br/>
-PageRecycleViewAdapter<br/>
-PageRecycleviewBaseAdapter<br/>

可直接使用原本的Adapter extends PageRecycleviewBaseAdapter<br/>
將原本的Adapter設定到PageRecycleViewAdapter內<br/>
可設定預設列數<br/>
